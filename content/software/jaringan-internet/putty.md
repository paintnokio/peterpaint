**PuTTY** adalah terminal emulator gratis untuk mengakses server melalui SSH, Telnet, dan lainnya.

### 🧩 Fitur:
- SSH key management
- Session logging
- Support SCP & serial

### 💻 Minimum:
- Windows / Linux
- RAM 512 MB
